# AI

AI endpoints for translating messages, paraphrasing thread titles, fixing grammar, and summarizing threads.

## Endpoints

<RouteHeader method="POST" url="/ai/translate">
  Translate Message
</RouteHeader>

Translates a message. Returns the translated message.

###### JSON Params

| Field   | Type   | Description                                                     |
| ------- | ------ | --------------------------------------------------------------- |
| content | string | The content of the message to be translated (1-2000 characters) |
| locale  | string | The locale to translate the message into (1-10 characters)      |

###### JSON Response

| Field       | Type   | Description                           |
| ----------- | ------ | ------------------------------------- |
| content ^1^ | string | The translated content of the message |

^1^ If the AI cannot translate the message, the original content will be returned.

<RouteHeader method="POST" url="/ai/title">
  Paraphrase Thread Title
</RouteHeader>

Paraphrases the title of a thread. Returns the paraphrased title.

###### JSON Params

| Field   | Type   | Description                                                      |
| ------- | ------ | ---------------------------------------------------------------- |
| content | string | The content of the message to be paraphrased (1-2000 characters) |

###### JSON Response

| Field | Type   | Description                         |
| ----- | ------ | ----------------------------------- |
| title | string | The paraphrased title of the thread |

<RouteHeader method="POST" url="/ai/fix-grammar">
  Fix Grammar
</RouteHeader>

Fixes the grammar of a message. Returns the message with corrected grammar.

###### JSON Params

| Field   | Type   | Description                                                    |
| ------- | ------ | -------------------------------------------------------------- |
| content | string | The content of the message to be corrected (1-2000 characters) |

###### JSON Response

| Field   | Type   | Description                        |
| ------- | ------ | ---------------------------------- |
| content | string | The message with corrected grammar |

<RouteHeader method="POST" url="/ai/summarize-thread/{thread.id}">
  Summarize Thread
</RouteHeader>

Summarizes the content of a thread. Returns a 204 empty response on success. The summary will be sent as a system message in the thread.

###### JSON Params

| Field      | Type    | Description                                                      |
| ---------- | ------- | ---------------------------------------------------------------- |
| ephemeral? | boolean | Whether the summary message should be ephemeral (default: false) |
