# Insights

Insights are a data monitoring and analytics tool that Discord provides to help community managers understand their guild's performance and how their members are interacting with it.

###### Aggregation Interval

| Value | Name    | Description |
| ----- | ------- | ----------- |
| 1     | DAILY   | Daily       |
| 2     | WEEKLY  | Weekly      |
| 3     | MONTHLY | Monthly     |

## Endpoints

###### Query String Params

<Alert type="info">

The following URL parameters are used in all endpoints that return insights data and do not change between requests.

</Alert>

| Field    | Type         | Description                                                 |
| -------- | ------------ | ----------------------------------------------------------- |
| start    | ISO8601 date | The start date for the insights data                        |
| end      | ISO8601 date | The end date for the insights data                          |
| interval | integer      | [The interval for the insights data](#aggregation-interval) |
| use_t2?  | boolean      | Unknown use case                                            |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/growth-activation/overview">
  Get Guild Growth Activation Overview
</RouteHeader>

Returns a list of [growth activation overview](#growth-activation-overview-structure) objects representing the amount of users who joined the guild, and the number of users who chatted in the guild.

###### Growth Activation Overview Structure

| Field             | Type         | Description                                             |
| ----------------- | ------------ | ------------------------------------------------------- |
| day_pt            | ISO8601 date | When the data was captured                              |
| new_members       | integer      | The number of new members who joined the guild          |
| new_communicators | integer      | The number of new members who communicated in the guild |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/growth-activation/joins">
  Get Guild Growth Activation Joins
</RouteHeader>

Returns a list of [growth activation joins](#growth-activation-joins-structure) objects that represents the number of users who joined the guild on a given date.

###### Growth Activation Joins Structure

| Field  | Type         | Description                                           |
| ------ | ------------ | ----------------------------------------------------- |
| day_pt | ISO8601 date | When the data was captured                            |
| joins  | integer      | The number of users who joined the guild on this date |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/growth-activation/joins-by-invite-link">
  Get Guild Growth Activation Joins by Invite
</RouteHeader>

Returns a list of [growth activation joins](#growth-activation-joins-by-invite-structure) objects that represents the number of users who joined the guild using an invite link.

###### Growth Activation Joins by Invite Structure

| Field       | Type         | Description                                                     |
| ----------- | ------------ | --------------------------------------------------------------- |
| day_pt      | ISO8601 date | The date the invite link was created                            |
| invite_link | string       | The invite link used to join the guild                          |
| joins       | integer      | The number of users who joined the guild using this invite link |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/growth-activation/joins-by-referrer">
  Get Guild Growth Activation Joins by Referrer
</RouteHeader>

Returns a list of [growth activation joins by referrer](#growth-activation-joins-by-referrer-structure) objects representing the number of users who joined the guild through website referrals like Google, Reddit, or other websites.

###### Growth Activation Joins by Referrer Structure

| Field                | Type         | Description                                                       |
| -------------------- | ------------ | ----------------------------------------------------------------- |
| day_pt               | ISO8601 date | The date the referral was made                                    |
| referring_domain ^1^ | string       | The referring domain that led to the referral link being clicked  |
| joins                | integer      | The number of users who joined the guild using this referral link |

^1^ The `referring_domain` field represents what the `Referer` header contained when they joined the guild.

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/growth-activation/joins-by-source">
  Get Guild Growth Activation Joins by Sources
</RouteHeader>

Returns a list of [growth activations joins by sources](#growth-activation-joins-by-sources-structure) objects representing the number of users who joined the guild by source, such as invites, integration, or other means.

###### Growth Activation Joins by Sources Structure

| Field             | Type         | Description                                                                                                                            |
| ----------------- | ------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| day_pt            | ISO8601 date | The date the source was recorded                                                                                                       |
| discovery_joins   | integer      | The total number of users who joined the guild through guild discovery                                                                 |
| invites           | integer      | The total number of users who joined the guild through an invite                                                                       |
| vanity_joins      | integer      | The total number of users who joined the guild through a vanity URL                                                                    |
| hubs_joins        | integer      | The total number of users who joined the guild through a student hub                                                                   |
| bot_joins         | integer      | The total number of users who were added to the guild by a bot using the [guilds.join OAuth2 scope](/resources/guild#add-guild-member) |
| integration_joins | integer      | The total number of users who were added to the guild by an integration (e.g. Twitch)                                                  |
| other_joins       | integer      | The total number of users who joined the guild through an unknown source                                                               |
| total_joins       | integer      | The total number of users who joined the guild                                                                                         |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/growth-activation/activation">
  Get Guild Growth Activation Percentages
</RouteHeader>

###### Growth Activation Percentages Structure

| Field               | Type         | Description                                                                                     |
| ------------------- | ------------ | ----------------------------------------------------------------------------------------------- |
| day_pt              | ISO8601 date | When the data was captured                                                                      |

| pct_communicated    | float        | The percentage of new members who sent 3 or more message in the guild or talked in a voice chat |
| pct_opened_channels | float        | The percentage of new members who opened 3 or more channels in the guild                        |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/engagement/base">
  Get Guild Engagement Base
</RouteHeader>

Returns a list of [engagement base](#engagement-base-structure) objects representing the number of users who visited the guild, communicated in the guild, and sent 3+ messages in the guild on a given date.

###### Engagement Base Structure

| Field                     | Type         | Description                                                       |
| ------------------------- | ------------ | ----------------------------------------------------------------- |
| day_pt                    | ISO8601 date | When the data was captured                                        |
| visitors                  | integer      | The number of users who visited the guild                         |
| communicators             | integer      | The number of users who sent 3 or more messages in the guild      |
| pct_communicators         | float        | The percentage of users who communicated in the guild             |
| messages                  | integer      | The number of messages sent in the guild on the date provided     |
| messages_per_communicator | float        | The average number of messages sent per communicator in the guild |
| speaking_minutes          | integer      | The total amount of time users spent talking in the guild         |

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/engagement/text-channels">
  Get Guild Engagement Text Channels
</RouteHeader>

Returns a list of [text channel engagement](#text-channel-engagement-structure) objects representing the number of users who communicated, and sent 3+ messages in a channel on a given date.

###### Text Channel Engagement Structure

| Field                            | Type         | Description                                                                    |
| -------------------------------- | ------------ | ------------------------------------------------------------------------------ |
| day_pt                           | ISO8601 date | When the data was captured                                                     |
| channel_name ^1^                 | string       | The name of the text channel                                                   |
| channel_id                       | snowflake    | The ID of the text channel                                                     |
| participators                    | integer      | The number of users who talked in the text channel                             |
| communicators                    | integer      | The number of users who talked in the text channel who sent 3 or more messages |
| messages_sent                    | integer      | The number of messages sent in the text channel                                |
| pct_participated_in_channel? ^2^ | float        | The percentage of users who participated in the text channel                   |
| pct_communicated_in_channel? ^2^ | float        | The percentage of users who communicated in the text channel                   |

^1^ The `channel_name` field is only present if the channel is a text channel. It is not present for voice channels.

^2^ The `pct_participated_in_channel` and `pct_communicated_in_channel` fields are only present if their respective values are greater than 0.

<RouteHeader method="GET" url="/guilds/{guild.id}/analytics/engagement/muters">
  Get Guild Engagement Muters
</RouteHeader>

Returns a list of [engagement muters](#engagement-muters-structure) objects representing the number of users who have muted the guild.

###### Engagement Muters Structure

| Field         | Type         | Description                                                                                                                                                                  |
| ------------- | ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| day_pt        | ISO8601 date | When the data was captured                                                                                                                                                   |
| days_in_guild | string       | Returns `Existing Members` if the user muted the guild only after a while, or `New Members` if the user muted the guild at the time they joined before the specified period. |
| muters        | integer      | The number of users who have muted the guild                                                                                                                                 |
