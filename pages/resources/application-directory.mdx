# Application Directory

The application directory allows users to find fun and entertaining applications with special filters to tailor their experience and the kind of applications that are shown.

### Category Object
###### Category Structure

| Field | Type    | Description                                   |
|-------|---------|-----------------------------------------------|
| id    | integer | The numerical ID associated with the category |
| name  | string  | The category label                            |

###### Category ID Type

| Value | Name                 | Description                                |
|-------|----------------------|--------------------------------------------|
| 4     | Entertainment        | Applications that offer entertainment      |
| 6     | Games                | Applications with games                    |
| 8     | Moderation and Tools | Applications with moderation and tools     |
| 9     | Social               | Applications with social features          |
| 10    | Utilities            | Applications that has a range of utilities |

### Directory Entry Object

###### Directory Entry Structure

| Field                            | Type                                                                                               | Description                                                    |
|----------------------------------|----------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| guild_count                      | integer                                                                                            | Approximate number of guilds the application is in             |
| detailed_description?            | string                                                                                             | The detailed overview of the application                       |
| carousel_items?                  | array[[carousel items](#carousel-item-structure) object]                                                                | The media to show with the application                         |
| supported_locales                | array[string]                                                                                      | The supported [locales](/reference#locales) by the application |
| external_urls?                   | array[[external url](#external-url-structure) object]                                              | External links related to the application                      |
| popular_application_command_ids? | array[string]                                                                                      | The IDs of most popular application commands                   |
| popular_application_commands?    | array[[application command](/interactions/application-commands#application-command-object) object] | The most popular application commands (max 5)                  |
| short_description                | string                                                                                             | The short overview of the application                          |

###### Carousel Item Structure

| Field     | Type    | Description                              |
|-----------|---------|------------------------------------------|
| type      | integer | Type of media (e.g. image, video)        |
| url       | string  | Source URL of the media                  |
| proxy_url | string  | A proxied URL of the media               |

###### Carousel Item Type Enum

| Value | Name         | Description                             |
|-------|--------------|-----------------------------------------|
| 1     | IMAGE        | Supports both static images and gifs    |
| 2     | VIDEO        | A video. Supports direct URL or YouTube |

###### External URL Structure

| Field | Type   | Description                       |
|-------|--------|-----------------------------------|
| name  | string | The name to display for the link  |
| url   | string | The URL to redirect the user to   |

###### Source Type

| Value | Name                         | Description                                 |
|-------|------------------------------|---------------------------------------------|
| 1     | APPLICATION_DIRECTORY        | User is coming from application directory   |
| 2     | APP_LAUNCHER_IN_TEXT         | User is coming from text based commands     |
| 3     | APP_LAUNCHER_IN_VOICE_BANNER | User is coming from a voice activity banner |

## Endpoints

<RouteHeader method="GET" url="/application-directory-static/categories">
  Get Application Categories
</RouteHeader>

Returns a list of [categories](#categroies-structure) objects representing the specific application categories.

###### Query String Params

| Field  | Type   | Description                                                |
|--------|--------|------------------------------------------------------------|
| locale | string | The [language](/reference#locales) to return categories in |

<RouteHeader method="GET" url="/application-directory-static/collections">
  Get Collections of Applications
</RouteHeader>

Returns application collections.

###### Query String Params

| Field            | Type    | Description                                                                                  |
|------------------|---------|----------------------------------------------------------------------------------------------|
| surface          | integer | [Where the request comes from](#source-type)                                                 |
| active_state ^1^ | integer | Filter applications by their [directory active state](#active-state-type) (default `ACTIVE`) |
| platform         | integer | The [platform](#platform-type-enum) the user is currently on                                                 |
| locale           | string  | The [language](/reference#locales) to return collections in                                  |
| cache ^1^        | boolean | Whether to return  cached results                                                            |

^1^ Only usable by Discord employees

###### Response Body

| Field                                  | Type                                                            | Description                                                  |
|----------------------------------------|-----------------------------------------------------------------|--------------------------------------------------------------|
| id                                     | string                                                          | The unique collection id                                     |
| type                                   | integer                                                         | ??                                                           |
| position                               | integer                                                         | The hierarchical order to make the collection appear in      |
| platforms                              | integer                                                         | The [platforms](#platform-type-enum) the collection supports                 |
| active_state                           | integer                                                         | Whether or not the collection is active                      |
| flags                                  | integer                                                         | ??                                                           |
| title                                  | string                                                          | The collection's title                                       |
| description                            | string                                                          | Short description of the collection                          |
| application_directory_collection_items | array[[application](/resources/application#application-object)]                                       | Applications that are in the collection                      |

###### Active State Type Enum

| Value | Name    | Description                          |
|-------|---------|--------------------------------------|
| 0     | PREVIEW | Inactive and hidden from public view |
| 1     | ACTIVE  | Active and can be seen by the public |

###### Platform Type Enum

| Value | Name    | Description                          |
|-------|---------|--------------------------------------|
| 1     | iOS     | The user is using a iOS device       |
| 2     | Android | The user is using a Android device   |
| 4     | Web     | The user is using a web client       |

<RouteHeader method="GET" url="/application-directory-static/applications/{application.id}">
  Get Application Details
</RouteHeader>

Returns an [application](/resources/application#application-object) object from the ID provided.

###### Query String Params

| Field   | Type    | Description                                                         |
|---------|---------|---------------------------------------------------------------------|
| locale  | string  | The [language](/reference#locales) to return application details in |
| nocache | boolean | Disables cache                                                      |

<RouteHeader method="GET" url="/application-directory-static/applications/{application.id}/similar">
  Get Similar Applications
</RouteHeader>

Returns a applications similar to it.

###### Query String Params

| Field    | Type    | Description                                                          |
|----------|---------|----------------------------------------------------------------------|
| guild_id | string  | The guild id of where the request came from                          |
| page     | integer | The page to be on                                                    |
| locale   | string  | The [language](/reference#locales) to return similar applications in |

###### Response Body

| Field        | Type                                                            | Description                                                           |
|--------------|-----------------------------------------------------------------|-----------------------------------------------------------------------|
| applications | array[[application](/resources/application#application-object)] | A list of applications that are similar to the current application    |
| num_pages    | integer                                                         | The number of pages containing similar applications                   |
| load_id      | string                                                          | A client-generated UUID used to identify the current checkout session |

<RouteHeader method="GET" url="/application-directory-static/search">
  Search For Applications
</RouteHeader>

Returns the search results.

###### Query String Params

| Field                                                         | Type    | Description                                                                       |
|---------------------------------------------------------------|---------|-----------------------------------------------------------------------------------|
| query                                                         | string  | What to search for                                                                |
| guild_id                                                      | string  | The guild id of where the request came from                                       |
| page                                                          | integer | The page to fetch results from                                                    |
| page_size                                                     | integer | The amount of entries to send                                                     |
| category_id                                                   | integer | The [category](#category-id-type) to find                                         |
| locale                                                        | string  | The [language](/reference#locales) to return search results in                    |
| min_user_install_command_count                                | integer | Specifies the minimum amount of user installs a application needs                 |
| exclude_apps_with_custom_install_url                          | boolean | Whether or not to exclude applications with custom install urls                   |
| exclude_non_embedded_apps                                     | boolean | Whether or not to exclude applications that are not embedded                      |
| exclude_embedded_apps_without_primary_entry_point_app_command | boolean | Whether or not to exclude applications that don't have a main entry point command |
| source                                                        | integer | [Where](#source-type-enum) the request came from                                  |

###### Results Structure

| Field | Type                                                            | Description                                |
|-------|-----------------------------------------------------------------|--------------------------------------------|
| type  | integer                                                         | Whether the application is hidden or not   |
| data  | [application](/resources/application#application-object) object | The application associated with the result |

###### Response Body

| Field              | Type                                 | Description                                                                                         |
|--------------------|--------------------------------------|-----------------------------------------------------------------------------------------------------|
| results            | [results](#results-structure) object | The results of the search                                                                           |
| num_pages          | integer                              | The number of pages the search yielded                                                              |
| counts_by_category | array[string, integer]               | The [category id](#category-id-type) along with the amount of applications that fit in the category |
| type               | integer                              | ??                                                                                                  |
| load_id            | string                               | A analytics UUID                                                                                    |
