# Application Directory

The application directory allows users to find fun and entertaining applications with special filters to tailor their experience and the kind of applications that are shown.

### Categories Object

###### Categories Structure

| Field | Type    | Description                                   |
| ----- | ------- | --------------------------------------------- |
| id    | integer | The numerical ID associated with the category |
| name  | string  | The category label                            |

###### Categories ID Enum

| Value | Name                 | Description                                                      |
| ----- | -------------------- | ---------------------------------------------------------------- |
| 4     | Entertainment        | Applications specializing in entertainment (e.g. Watch Together) |
| 6     | Games                | Applications specializing in games                               |
| 8     | Moderation and Tools | Applications specializing in moderation                          |
| 9     | Social               | Applications spealizing in socialness                            |
| 10    | Utilities            | Applications specializing with utilities                         |

### Directory Entry Object

###### Directory Entry Structure

| Field                            | Type                                                                    | Description                                                      |
| -------------------------------- | ----------------------------------------------------------------------- | ---------------------------------------------------------------- |
| guild_count                      | integer                                                                 | Specifies the amount of guilds a application is in               |
| detailed_description?            | string                                                                  | The long 'overview' part of the application                      |
| carousel_items?                  | array[[carousel items](#carousel-items-structure) object]               | The media to add to the application page                         |
| supported_locales                | array[[locale](/reference#locales)]                                     | Specifies the supported locales by the application               |
| external_urls?                   | array[[external urls](#external-urls-structure) object]                 | Special links to add to the application's page                   |
| popular_application_command_ids? | array[string]                                                           | The most popular application commands IDs                        |
| popular_application_commands?    | array[[application command](/interactions/application-commands) object] | The most popular application commands                            |
| short_description                | string                                                                  | A short and sweet basic description on what the application does |

###### Carousel Items Structure

| Field     | Type    | Description                              |
| --------- | ------- | ---------------------------------------- |
| type      | integer | Type of media (e.g. image, video)        |
| url       | string  | Direct link to the media source          |
| proxy_url | string  | Discord CDN proxied version of the media |

###### Carousel Item Type Enum

| Value | Name      | Description                             |
| ----- | --------- | --------------------------------------- |
| 1     | Image/GIF | Supports both static images and gifs    |
| 2     | Video     | A video. Supports direct URL or YouTube |

###### External URLs Structure

| Field | Type   | Description                       |
| ----- | ------ | --------------------------------- |
| name  | string | The name to display for the link  |
| url   | string | The place to redirect the user to |

## Endpoints

<RouteHeader method="GET" url="/application-directory-static/categories">
  Get Application Categories
</RouteHeader>

Returns a list of [categories](#categroies-structure) objects representing the specific application categroies.

###### Query String Params

| Field  | Type   | Description                                                                      |
| ------ | ------ | -------------------------------------------------------------------------------- |
| locale | string | The [locale](/reference#locales) to translate the message into (1-10 characters) |

<RouteHeader method="GET" url="/application-directory-static/collections">
  Get Collections of Applications
</RouteHeader>

Returns a list of [application](/resources/application#application-object) objects that fits the filters applied.

###### Query String Params

| Field           | Type    | Description                                                                      |
| --------------- | ------- | -------------------------------------------------------------------------------- |
| surface         | integer | ??                                                                               |
| active_state^1^ | integer | Whether or not to show inactive applications (0 for inactive, 1 for active)      |
| platform        | integer | The platform the user is currently on (1 iOS, 2 Android, 4 Web)                  |
| locale          | string  | The [locale](/reference#locales) to translate the message into (1-10 characters) |
| cache^1^        | bool    | Enables or disables cached results                                               |

^1^ Staff only option.

<RouteHeader method="GET" url="/application-directory-static/applications/{application.id}">
  Get Application Details
</RouteHeader>

Returns a [application](/resources/application#application-object) object from the ID provided.

###### Query String Params

| Field  | Type   | Description                                                                      |
| ------ | ------ | -------------------------------------------------------------------------------- |
| locale | string | The [locale](/reference#locales) to translate the message into (1-10 characters) |

<RouteHeader method="GET" url="/application-directory-static/applications/{application.id}/similar">
  Get Similar Applications
</RouteHeader>

Returns a [similar](#similar-structure) object.

###### Query String Params

| Field  | Type   | Description                                                                      |
| ------ | ------ | -------------------------------------------------------------------------------- |
| locale | string | The [locale](/reference#locales) to translate the message into (1-10 characters) |

###### Similar Structure

| Field        | Type                                                            | Description                                                           |
| ------------ | --------------------------------------------------------------- | --------------------------------------------------------------------- |
| applications | array[[application](/resources/application#application-object)] | A list of applications that are similar to the current application    |
| num_pages    | integer                                                         | The number of pages containing similar applications                   |
| load_id      | string                                                          | A client-generated UUID used to identify the current checkout session |

<RouteHeader method="GET" url="/application-directory-static/search">
  Search For Applications
</RouteHeader>

Returns a [search](#search-structure) object.

###### Query String Params

| Field       | Type    | Description                                                                      |
| ----------- | ------- | -------------------------------------------------------------------------------- |
| query       | string  | What to search for                                                               |
| page        | integer | The page to fetch results from                                                   |
| page_size   | integer | The amount of entries to send                                                    |
| category_id | integer | The [category](#categories-id-enum) to find                                      |
| locale      | string  | The [locale](/reference#locales) to translate the message into (1-10 characters) |

###### Results Structure

| Field | Type                                                            | Description                                |
| ----- | --------------------------------------------------------------- | ------------------------------------------ |
| type  | integer                                                         | Whether the application is hidden or not   |
| data  | [application](/resources/application#application-object) object | The application associated with the result |

###### Search Structure

| Field              | Type                                 | Description                                                                                           |
| ------------------ | ------------------------------------ | ----------------------------------------------------------------------------------------------------- |
| results            | [results](#results-structure) object | The results of the search                                                                             |
| num_pages          | integer                              | The number of pages the search yielded                                                                |
| counts_by_category | array[string, integer]               | The [category id](#categories-id-enum) along with the amount of applications that fit in the category |
| type               | integer                              | ??                                                                                                    |
| load_id            | string                               | A client-generated UUID used to identify the current checkout session                                 |
