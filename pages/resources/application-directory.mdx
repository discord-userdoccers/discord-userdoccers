# Application Directory

The application directory allows users to find fun and entertaining applications with special filters to tailor their experience and the kind of applications that are shown.

### Category Object

###### Category Structure

| Field | Type    | Description                                   |
| ----- | ------- | --------------------------------------------- |
| id    | integer | The numerical ID associated with the category |
| name  | string  | The category label                            |

###### Category ID Type

| Value | Name                 | Description                                |
| ----- | -------------------- | ------------------------------------------ |
| 4     | Entertainment        | Applications that offer entertainment      |
| 6     | Games                | Applications with games                    |
| 8     | Moderation and Tools | Applications with moderation and tools     |
| 9     | Social               | Applications with social features          |
| 10    | Utilities            | Applications that has a range of utilities |

### Directory Entry Object

###### Directory Entry Structure

| Field                            | Type                                                                                               | Description                                                            |
| -------------------------------- | -------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| guild_count                      | integer                                                                                            | Approximate number of guilds the application is in                     |
| carousel_items?                  | array[[carousel items](#carousel-item-structure) object]                                           | The media to show with the application                                 |
| supported_locales                | array[string]                                                                                      | The [locales](/reference#locales) supported by the application         |
| external_urls?                   | array[[external url](#external-url-structure) object]                                              | External links related to the application                              |
| popular_application_command_ids? | array[string]                                                                                      | The IDs of the application's most popular application commands (max 5) |
| popular_application_commands?    | array[[application command](/interactions/application-commands#application-command-object) object] | The application's most popular application commands (max 5)            |
| detailed_description?            | string                                                                                             | The detailed overview of the application                               |
| short_description                | string                                                                                             | The short overview of the application                                  |

###### Carousel Item Structure

| Field     | Type    | Description                          |
| --------- | ------- | ------------------------------------ |
| type      | integer | [Type](#carousel-item-type) of media |
| url       | string  | Source URL of the media              |
| proxy_url | string  | A proxied URL of the media           |

###### Carousel Item Type

| Value | Name  | Description                             |
| ----- | ----- | --------------------------------------- |
| 1     | IMAGE | Supports both static images and gifs    |
| 2     | VIDEO | Supports both static videos and YouTube |

###### External URL Structure

| Field | Type   | Description                      |
| ----- | ------ | -------------------------------- |
| name  | string | The name to display for the link |
| url   | string | The URL to redirect the user to  |

###### Request Surface

| Value | Name                         | Description                         |
| ----- | ---------------------------- | ----------------------------------- |
| 1     | APPLICATION_DIRECTORY        | Application directory               |
| 2     | APP_LAUNCHER_IN_TEXT         | App Launcher                        |
| 3     | APP_LAUNCHER_IN_VOICE_BANNER | App Launcher within a voice context |

## Endpoints

<RouteHeader method="GET" url="/application-directory-static/categories" unauthenticated>
  Get Application Directory Categories
</RouteHeader>

Returns a list of [categories](#categories-structure) objects representing the available application categories.

###### Query String Params

| Field   | Type   | Description                                                                  |
| ------- | ------ | ---------------------------------------------------------------------------- |
| locale? | string | The [language](/reference#locales) to return categories in (default "en-US") |

<RouteHeader method="GET" url="/application-directory-static/collections" unauthenticated>
  Get Application Direction Collections
</RouteHeader>

Returns the available application directory collections.

###### Query String Params

| Field            | Type    | Description                                                                                  |
| ---------------- | ------- | -------------------------------------------------------------------------------------------- |
| surface          | integer | The [request surface](#request-surface), used for analytics                                  |
| active_state ^1^ | integer | Filter applications by their [directory active state](#active-state-type) (default `ACTIVE`) |
| platform         | integer | The [platform](#platform-type) the user is currently on                                      |
| locale           | string  | The [language](/reference#locales) to return collections in                                  |
| cache ^1^        | boolean | Whether to return cached results                                                             |

^1^ Only usable by Discord employees.

###### Response Body

| Field                                  | Type                                                            | Description                                             |
| -------------------------------------- | --------------------------------------------------------------- | ------------------------------------------------------- |
| id                                     | string                                                          | The ID of the collection                                |
| type                                   | integer                                                         | ??                                                      |
| position                               | integer                                                         | The hierarchical order to make the collection appear in |
| platforms                              | integer                                                         | The [platforms](#platform-type) the collection supports |
| active_state                           | integer                                                         | Whether or not the collection is active                 |
| flags                                  | integer                                                         | ??                                                      |
| title                                  | string                                                          | The collection's title                                  |
| description                            | string                                                          | Short description of the collection                     |
| application_directory_collection_items | array[[application](/resources/application#application-object)] | Applications that are in the collection                 |

###### Active State Type

| Value | Name    | Description                          |
| ----- | ------- | ------------------------------------ |
| 0     | PREVIEW | Inactive and hidden from public view |
| 1     | ACTIVE  | Active and can be seen by the public |

###### Platform Type

| Value | Name    | Description                        |
| ----- | ------- | ---------------------------------- |
| 1     | iOS     | The user is using a iOS device     |
| 2     | Android | The user is using a Android device |
| 4     | Web     | The user is using a web client     |

<RouteHeader method="GET" url="/application-directory-static/applications/{application.id}" unauthenticated>
  Get Application Directory Application
</RouteHeader>

Returns a partial [application](/resources/application#application-object) object for the given ID.

###### Query String Params

| Field   | Type    | Description                                                         |
| ------- | ------- | ------------------------------------------------------------------- |
| locale  | string  | The [language](/reference#locales) to return application details in |
| nocache | boolean | Whether to bypass cache for the response (default false)            |

<RouteHeader method="GET" url="/application-directory-static/applications/{application.id}/similar" unauthenticated>
  Get Application Directory Similar Applications
</RouteHeader>

Returns applications similar to the given application ID.

###### Query String Params

| Field    | Type    | Description                                                          |
| -------- | ------- | -------------------------------------------------------------------- |
| guild_id | string  | The ID of the guild the request originated from                      |
| page     | integer | The page to be on                                                    |
| locale   | string  | The [language](/reference#locales) to return similar applications in |

###### Response Body

| Field        | Type                                                            | Description                                                         |
| ------------ | --------------------------------------------------------------- | ------------------------------------------------------------------- |
| applications | array[[application](/resources/application#application-object)] | A list of applications that are similar to the current application  |
| num_pages    | integer                                                         | The number of pages containing similar applications                 |
| load_id      | string                                                          | The unique identifier for the application directory recommendations |

<RouteHeader method="GET" url="/application-directory-static/search" unauthenticated>
  Search Applications Directory
</RouteHeader>

Returns the application directory search results.

###### Query String Params

| Field                                                         | Type    | Description                                                                                             |
| ------------------------------------------------------------- | ------- | ------------------------------------------------------------------------------------------------------- |
| query                                                         | string  | What to search for                                                                                      |
| guild_id                                                      | string  | The guild id of where the request came from                                                             |
| page                                                          | integer | The page to fetch results from                                                                          |
| page_size                                                     | integer | The limit of items per page (1-100)                                                                     |
| category_id                                                   | integer | The [category](#category-id-type) to filter results by                                                  |
| locale                                                        | string  | The [language](/reference#locales) to return search results in                                          |
| min_user_install_command_count                                | integer | The minimum amount of application user installs to filter by                                            |
| exclude_apps_with_custom_install_url                          | boolean | Whether to exclude applications with `custom_install_url`                                               |
| exclude_non_embedded_apps                                     | boolean | Whether to exclude applications without the [`EMBEDDED` flag](/resources/application#application-flags) |
| exclude_embedded_apps_without_primary_entry_point_app_command | boolean | Whether to exclude embedded applications without a primary entry point command                          |
| source                                                        | integer | [Where](#source-type) the request came from                                                             |

###### Application Directory Search Results Structure

| Field | Type                                                            | Description                                |
| ----- | --------------------------------------------------------------- | ------------------------------------------ |
| type  | integer                                                         | Whether the application is hidden or not   |
| data  | [application](/resources/application#application-object) object | The application associated with the result |

###### Response Body

| Field              | Type                                 | Description                                                |
| ------------------ | ------------------------------------ | ---------------------------------------------------------- |
| results            | [results](#results-structure) object | The results of the search                                  |
| num_pages          | integer                              | The total number of pages the search yielded               |
| counts_by_category | map[integer, integer]                | Total results per directory category ID                    |
| type               | integer                              | ??                                                         |
| load_id            | string                               | The unique identifier for the application directory search |
