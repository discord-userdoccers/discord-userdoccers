# Wishlists

### Wishlist Object

###### Wishlist Structure

| Field              | Type                                                                           | Description                                 |
| ------------------ | ------------------------------------------------------------------------------ | ------------------------------------------- |
| id                 | snowflake                                                                      | The ID of the wishlist                      |
| user_id            | snowflake                                                                      | The ID of the user                          |
| wishlist_items     | array[[wishlist item](#wishlist-item-object) object]                           | The items in the wishlist                   |
| applications       | array[partial [application](/resources/application#application-object) object] | The applications referenced in the wishlist |
| storefront_pricing | [storefront pricing](/resources/store#storefront-pricing-object) object        | The storefront pricing of the wished items  |

### Wishlist Item Object

###### Wishlist Item Structure

| Field              | Type                                                                               | Description                                                                      |
| ------------------ | ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| sku_id             | snowflake                                                                          | The ID of the SKU                                                                |
| sku_product_line   | integer                                                                            | The [product line](/resources/store#sku-product-line) the SKU belongs to         |
| sku_name           | string                                                                             | The name of the SKU                                                              |
| is_owned?          | boolean                                                                            | Whether the item is owned by the user                                            |
| collectibles_item? | ?[collectible item](/resources/collectibles#collectible-item-object) object        | The collectible item that the user wants to get                                  |
| bundle_items?      | ?array[[collectible item](/resources/collectibles#collectible-item-object) object] | The collectible items that are included when the user gets the wished bundle SKU |   
| sku                | [SKU](/resources/store#sku-object) object                                          | The SKU that the user wants to get                                               |

###### Wishlist Visibility

| Value | Name    | Description             |
| ----- | ------- | ----------------------- |
| 0     | PRIVATE | The wishlist is private |
| 1     | PUBLIC  | The wishlist is pubilc  |

## Endpoints

<RouteHeader method="GET" url="/wishlists/{wishlist.id}">
  Get Wishlist
</RouteHeader>

Returns a [wishlist](#wishlist-object) object for the given wishlist ID.

###### Query String Params

| Field            | Type    | Description                                                                                       |
| ---------------- | ------- | ------------------------------------------------------------------------------------------------- |
| country_code?    | string  | The user's [ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code    |
| payment_gateway? | integer | The [payment gateway](/resources/billing#payment-gateway) (only `APPLE` and `GOOGLE` are allowed) |

<RouteHeader method="PATCH" url="/users/@me/wishlists/{wishlist.id}">
  Modify Wishlist
</RouteHeader>

Modifies a wishlist. Returns an updated [wishlist](#wishlist-object) object on success.

###### JSON Params

| Field            | Type    | Description                                                                                       |
| ---------------- | ------- | ------------------------------------------------------------------------------------------------- |
| visibility?      | integer | The [visibility of wishlist](/resources/wishlist#wishlist-visibility)                             |
| country_code?    | string  | The user's [ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code    |
| payment_gateway? | integer | The [payment gateway](/resources/billing#payment-gateway) (only `APPLE` and `GOOGLE` are allowed) |

<RouteHeader method="POST" url="/users/@me/wishlist/items">
  Add Wishlist Item
</RouteHeader>

Adds an item to the wishlist. Returns an updated [wishlist](#wishlist-object) object on success.

###### JSON Params

| Field            | Type      | Description                                                                                       |
| ---------------- | --------- | ------------------------------------------------------------------------------------------------- |
| sku_id           | snowflake | The ID of the SKU to add to the wishlist                                                          |
| country_code?    | string    | The user's [ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code    |
| payment_gateway? | integer   | The [payment gateway](/resources/billing#payment-gateway) (only `APPLE` and `GOOGLE` are allowed) |

<RouteHeader method="PATCH" url="/users/@me/wishlists/{wishlist.id}/items/{sku.id}">
  Reposition Wishlist Item
</RouteHeader>

Repositions the wishlist item near specified SKU IDs. Returns an updated [wishlist](#wishlist-object) object on success, or 204 empty response if both `previous_sku_id`, and `next_sku_id` were null.

###### JSON Params

| Field            | Type       | Description                                                                                       |
| ---------------- | ---------- | ------------------------------------------------------------------------------------------------- |
| previous_sku_id  | ?snowflake | The ID of the SKU to reposition the wishlist item after                                           |
| next_sku_id      | ?snowflake | The ID of the SKU to reposition the wishlist item before                                          |
| country_code?    | string     | The user's [ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code    |
| payment_gateway? | integer    | The [payment gateway](/resources/billing#payment-gateway) (only `APPLE` and `GOOGLE` are allowed) |

<RouteHeader method="DELETE" url="/users/@me/wishlists/{wishlist.id}/items/{sku.id}">
  Remove Wishlist Item
</RouteHeader>

Removes a wishlist item for the given wishlist and SKU ID. Returns a [wishlist](#wishlist-object) object on success.

###### Query String Params

| Field            | Type    | Description                                                                                       |
| ---------------- | ------- | ------------------------------------------------------------------------------------------------- |
| country_code?    | string  | The user's [ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code    |
| payment_gateway? | integer | The [payment gateway](/resources/billing#payment-gateway) (only `APPLE` and `GOOGLE` are allowed) |
